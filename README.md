# Практична робота "Масиви, вирази, керування виконанням програми"

Цей репозиторій містить стартовий код для виконання практичної роботи, який містить заготовки двох класів, що демонструють структуру проекту для виконання завдань на масиви, керуючі конструкції та вирази.

## Було обране завдання: 
Відсортувати одновимірний масив, розбиваючи його на групи елементів по 1, 2, 4, 8 і т.д. Дозволяється використовувати допоміжний масив

В рамках практичної роботи було реалізовано алгоритм ітеративного сортування масиву шляхом послідовного злиття груп елементів розміром [1,2,4,8] і тд. Основна логіка винесена в клас ```Exercise```, де для оптимізації пам'яті використовується один допоміжний масив-буфер. Проєкт структуровано за пакетами, додано Javadoc-коментарі для методів та налаштовано тестовий запуск через клас ```TestResult``` для перевірки коректності роботи алгоритму на довільних вхідних даних.

## Запуск програми
Щоб запустити програму треба в консолі ввести цю команду: java -cp src test.TestResult
